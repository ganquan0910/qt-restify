# Restify

An application for visualizing and testing RESTful requests.

## Compiling on Mac OS X

1. Install Xcode with Command Line Tools
2. Install Qt libraries 4.8.3 for Mac
3. In terminal:

`cd restify
qmake -project
qmake -spec macx-g++
make`
